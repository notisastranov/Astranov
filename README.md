**Internet Operating System**

Astranov is an internet operating system where the Earth itself is the interface.

## Trinity Layers

| Layer | Scene | HUD Theme |
|---|---|---|
| L1 Cosmic | Three.js Globe + Starfield | Spacecraft — electric blue |

| L3 Abyssal | Deep Core + Particles | Submarine — deep blue |

---

## Stack

- **Frontend:** Vanilla HTML/CSS/JS — Three.js, Leaflet
- **Backend:** Express + Node.js
- **Database:** MySQL
- **Maps:** OpenStreetMap (no Google Maps dependency)
- **Deployment:** Vercel + Cloudflare
- **Architecture:** Spartan Script v3

---

## Deployment

```bash
# Clone
git clone https://github.com/notisastranov/Astranov.git
cd Astranov

# Deploy to Vercel
npx vercel --prod
```

Point `astranov.eu` DNS to Vercel nameservers via Cloudflare.
