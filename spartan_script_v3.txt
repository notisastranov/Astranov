SPARTAN SCRIPT v3 — ASTRANOV TRINITY OS
Creator: Notis Astranov, 2026. Method: Spartan Script.

IDENTITY
Astranov is a celestial OS descending through three nested realities:
L1 COSMIC: orbital space, globe, spacecraft HUD.
L2 TERRESTRIAL: Earth surface, city map, commerce HUD.
L3 ABYSSAL: deep core, sonar, submarine HUD.
One locked shell. Three adaptive scenes.

SHELL — INVARIANT
Grid: top-strip(28px) / [left-hud(48px) | stage | right-hud(48px)] / ai-bar(50px).
Fixed. Nothing moves. Layout never changes between layers. Colors adapt.

TOP STRIP
Centered: status-dot | ASTRANOV (Orbitron, accent-glow) | layer-badge | current vX.X.X | latest vX.X.X (green=match, red=mismatch + yellow update link).
Forbidden: circles, capsules, diagnostics, extra items.

LEFT HUD (registry only)
Locked order: settings → profile → diagnostics → console → [spacer] → POST (permanent, bottom).
No unapproved buttons. No About/Help/Credits.

RIGHT HUD (registry only)
Locked order: layers → filters → GPS → signals.
No duplicate toggles. No rogue buttons.

AI BAR (bottom-center)
Left: voice-toggle, camera. Center: text input. Right: mic, send.
Sends POST /ai-command {message, coordinates, layer, context}.
Backend handles AI. Frontend never calls provider directly.

RADAR (bottom-right)
Toggle: open/close rectangular panel anchored below button.
No float. No overlap with AI bar.

MENUS
Rectangular. Semi-transparent. Anchored to source button. Source stays highlighted while open.

LAYER SYSTEM
currentLayer in {0,1,2}. Transition: 150ms flash → swap body class → swap canvas → update theme.
body.layer-cosmic: --layer-accent:#00c8ff.
body.layer-terrestrial: --layer-accent:#00e87a.
body.layer-abyssal: --layer-accent:#4466ff.
All UI elements inherit accent via CSS vars.

L1 COSMIC
Three.js. SphereGeometry(1,48,48). MeshPhongMaterial(canvasTexture, shininess 14).
Atmosphere: ShaderMaterial rim (AdditiveBlending, FrontSide).
Stars: Points(3000, size 0.06). Signals: Points(80, vertexColors, pulse opacity+size).
Rotation y+=0.0007/frame + drag inertia. Camera z=2.6.

L2 TERRESTRIAL
Leaflet + OSM tiles. Filter: brightness(0.75) saturate(0.6) hue-rotate(120deg).
Signal markers: DivIcon 10px, category colors:
commerce=#ffd600, social=#00c8ff, jobs=#00e87a, news=#ff4455, dating=#ff80ab, real_estate=#c084fc.
Map click emits coords to AI context.

L3 ABYSSAL
Three.js. IcosahedronGeometry(1.2,4). MeshPhongMaterial(color 0x001a4d, emissive 0x0a1a4f).
Two TorusGeometry rings counter-rotating. 800 blue particles. PointLight orbiting.

BACKEND
Frontend: render HUD, collect input, send requests, render responses only.
Backend: /ai-command /geo-action /business-action /order-action /delivery-action.
Stack: Express + MySQL + WebSocket. No Google/Firebase/Gemini.

ANTI-DRIFT
No rogue HUD buttons. No dashboard drift. No diagnostics in strip.
Center stage = scene only. 1 command = 1 subsystem = 1 result.
